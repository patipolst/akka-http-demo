# Akka HTTP Demo

Simple Akka-http REST API example modified and inspired from [ArchDev/akka-http-rest](https://github.com/ArchDev/akka-http-rest/)

## Features
* Based on Akka HTTP
* Database connection with Slick
* Using Circe Json implementation
* Case class validation handle by Accord
* Authentication with JWT
* CORS Support
* Tests coverage with Scalatest
